# Pods and Namespaces

## Pods
    - smallest deployable unit in Kubernetes
    - can contain multiple containers
    - share network and storage

## Pod Lifecycle
    - Pending
    - Running
    - CrashLoopBackOff
    - OOMKilled
    - Completed

## Namespaces
    - provide logical isolation
    - enables multi-tenancy
    - used heavily in Openshift environments
